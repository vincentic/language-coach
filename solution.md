# 通识教育 × 生活实用 知识漏洞分析与补全策略

---

## 一、现状总览

| 指标 | 数值 |
|------|------|
| 总知识节点 | 13,957 |
| 总关系边 | 15,236 |
| 唯一分类数 | 2,015 |
| 数据来源 | DeepSeek (1,819), Kimi (22), ChatGPT (3), Claude (2) |

**核心问题：分类碎片化** — 2,015 个分类中，1,127 个（56%）仅有 1-2 个节点。AI 提取时自由生成分类，导致同一学科被拆成数十个微分类。

## 二、双层知识框架

```
┌─────────────────────────────────────────────────────────┐
│                    通识教育（学术基础）                      │
│  哲学 · 历史 · 文学 · 数学 · 科学 · 社会 · 心理 · 法学     │
│  医学 · 工程 · 商业 · 艺术 · 教育 · 地理 · 政治            │
├─────────────────────────────────────────────────────────┤
│                    生活实用（日常应用）                      │
│  健康 · 财务 · 教育 · 事业 · 法律 · 时间 · 关系沟通 · 知识   │
└─────────────────────────────────────────────────────────┘
```

**关系**：通识教育是"底层操作系统"，生活实用是"应用层"。每个生活领域都由多个通识学科支撑。
