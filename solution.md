#### 🔥 亲密关系深度框架（源自 intimacy.md）

> 真正的亲密关系，是两个独立完整的人，选择走在一起，共同面对人生的旅程。
> 本质是：自我成长 × 共同成长。

**六大核心能力**：

| 能力 | 核心命题 | 关键实践 |
|------|---------|---------|
| 🪞 洞察 | 你爱上的是真实的人，还是想象中的幻影？ | 放下"应该"思维；观察对方真实行为；不试图改变对方 |
| ⚖️ 价值观 | 你们的方向是否一致？ | 深度对话而非假设；找交集而非强求一致；尊重差异 |
| 🦋 独立 | 没有对方，你是否依然是完整的你？ | 保持自我空间；不把幸福外包；学会自我满足 |
| 🛡️ 责任 | 你在为这段关系贡献什么？ | 承担自己的部分；不指责不逃避；看见自己的模式 |
| 🚧 边界 | 你在保护自己的同时，是否也为爱留了入口？ | 明确底线并坚持；不牺牲自我换和平；学会说"不" |
| 💬 需求 | 你能不能不带攻击性地说出"我需要你"？ | 用"我"语句而非"你"语句；区分需要和控制 |

**关系成熟度模型**：

```
Level 1: 依赖 — "没有你我活不下去"
Level 2: 独立 — "我可以没有你"
Level 3: 互赖 — "我选择和你在一起"
Level 4: 共创 — "我们在一起，创造更大的可能"
```

**亲密关系成长路径**：
```
阶段1 独立 → 认识自己、明确价值观、学会自我满足
阶段2 相遇 → 展示真实、用心观察、深度对话
阶段3 磨合 → 看见差异、学会沟通、建立规则
阶段4 共生 → 互相支持、共同成长、创造意义
```

**关系中的核心纪律**：

| 做 ✅ | 不做 ❌ |
|------|--------|
| 我为自己的感受负责 | 我不为对方的情绪买单 |
| 我用"我"语句表达需求 | 我不用"你总是"来指责 |
| 我相信行动而非承诺 | 我不为潜力而留下 |
| 我直面冲突，修复连接 | 我不通过第三方传话 |
| 我接受"足够好"的关系 | 我不追求完美无瑕的童话 |

**六个真相**：
1. 真正的亲密，是敢于展示脆弱
2. 好的关系不是没有冲突，而是冲突后能修复
3. 你不需要被所有人喜欢
4. 独立不是不需要别人，而是不依赖别人
5. 真正的爱，是给对方自由，而不是束缚
6. 成长是痛苦的，但不成长更痛苦
